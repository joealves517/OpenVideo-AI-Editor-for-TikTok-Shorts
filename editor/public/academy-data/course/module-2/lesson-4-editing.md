# Lesson 4: Video Editing & Customization

**Duration**: 30 minutes  
**Goal**: Master video editing to perfect your content

---

## Why Editing Matters

**The Problem**:
- First generation might not be perfect
- Need to adjust visuals
- Want to add personal touches
- Improve quality

**The Solution: Full Editing Suite**:
- Edit images and clips
- Customize subtitles
- Add motion and effects
- Perfect your content

**Editing can improve video performance by 20-30%.**

---

## Accessing the Video Editor

### How to Access

**Method 1: From My Videos**:
1. Go to "My Videos" section
2. Find video you want to edit
3. Click "Edit Images, Subtitles"
4. Editing panel opens

**Method 2: After Generation**:
1. Video appears in "My Videos"
2. Click "Edit" immediately
3. Make adjustments
4. Save changes

**Pro Tip**: Edit videos while they're fresh in your mind.

---

## Editing Images and Clips

### Understanding the Interface

**What You See**:
- All images used in video
- Each image paired with script segment
- Easy to identify what needs editing
- Clear organization

**How It Works**:
- Images listed in order
- Script segment shown for each
- Click "Edit Image" to modify
- Multiple editing options

---

### Five Powerful Editing Options

#### Option 1: Add Motion

**What It Does**:
- Animates existing images
- Replaces with motion-based scenes
- Brings static images to life
- Creates dynamic visuals

**How to Use**:
1. Click "Edit Image"
2. Select "Add Motion"
3. Enter motion prompt (e.g., "make fire burn, stars move")
4. Generate motion
5. Save

**When to Use**:
- Static images need movement
- Want more dynamic visuals
- Key moments in video
- Important scenes

**Cost**: **100% Free & Unlimited** (runs client-side with WebCodecs acceleration)

**Pro Tip**: Add motion to key moments for maximum impact.

---

#### Option 2: Upload Image

**What It Does**:
- Bring in your own media
- Use custom images/photos
- Full control over visuals
- Personal branding

**How to Use**:
1. Click "Edit Image"
2. Select "Upload Image"
3. Choose file from device
4. Upload
5. Save

**When to Use**:
- Have specific images
- Want custom branding
- Product photos
- Personal assets

**Cost**: **FREE** - Upload directly from your device

**Pro Tip**: Build library of custom images for reuse.

---

#### Option 3: Image Regenerate

**What It Does**:
- Creates new image with new prompt
- Multiple options to choose from
- Fresh visuals
- Better matches

**How to Use**:
1. Click "Edit Image"
2. Select "Image Regenerate"
3. Enter new prompt
4. Review options
5. Select favorite
6. Save

**When to Use**:
- Image doesn't match script
- Want different style
- Need better quality
- Testing variations

**Cost**: **FREE & Unlimited**

**Pro Tip**: Regenerate as many times as you like until you get the perfect image for your script.

---

#### Option 4: Stock Videos

**What It Does**:
- Access full stock video library
- Search by keyword
- Professional footage
- Perfect matches

**How to Use**:
1. Click "Edit Image"
2. Select "Stock Videos"
3. Search keyword
4. Browse options
5. Select clip
6. Save

**When to Use**:
- Need professional footage
- Want variety
- Specific scenes needed
- B-roll content

**Cost**: **FREE** - No credit cost

**Pro Tip**: Stock videos add professional polish.

---

#### Option 5: Asset Library

**What It Is**: Your personal collection of previously created content

**What's Included**:
- Previously generated images
- Uploaded files
- Used stock clips
- All your assets

**How to Use**:
1. Click "Edit Image"
2. Select "Asset Library"
3. Browse your assets
4. Select what you need
5. Save

**When to Use**:
- Reuse previous content
- Maintain consistency
- Save creation time
- Build brand assets

**Cost**: **FREE** - Saved locally in your browser database

**Pro Tip**: Asset library grows with every video - use it!

---

## Editing Subtitles

### Subtitle Customization

**What You Can Edit**:
- Subtitle text
- Font style
- Position (top, middle, bottom)
- Animation
- Colors

**How to Edit**:
1. Go to subtitle section
2. Click on subtitle
3. Edit text or style
4. Save changes

**Important Note**: 
- Changing subtitle text **won't affect voiceover**
- To update voice, edit script and regenerate video

**Pro Tip**: Subtitles are great for SEO - make them accurate.

---

### Advanced Subtitle Options

**Positioning**:
- Top: Good for landscape videos
- Middle: Avoid (blocks content)
- Bottom: Standard, recommended

**Styles**:
- Gummy (popular)
- Modern
- Classic
- Custom

**Animations**:
- Slide in
- Fade in
- Typewriter
- None

**Pro Tip**: Test different styles to see what works for your audience.

## Saving Your Edits

### Save Process

**How to Save**:
1. Make all edits
2. Review changes
3. Click "Save Video"
4. New version created
5. Original preserved

**What Happens**:
- Changes saved as new version
- Original video untouched
- Can revert if needed
- Export is 100% FREE and processed client-side

**Pro Tip**: Always review before saving.

---

## Pro Tips for Editing

### 1. Edit Strategically

**Focus On**:
- Key moments
- Important visuals
- Hook scenes
- CTA moments

**Don't Over-Edit**:
- Every image doesn't need editing
- Focus on what matters
- Done is better than perfect

---

### 2. Build Asset Library

**Strategy**:
- Save good images
- Reuse across videos
- Build brand assets
- Build consistency

**Result**: Free content library that grows.

---

### 3. Leverage Local Rendering

**Client-Side Rendering Benefits**:
- Unlimited video exports
- Dynamic asset library integration
- Instant local media uploads
- High-quality stock library integration

**Maximize**: Reuse your local asset library to build videos without re-downloading large files.

---

### 4. Test and Learn

**Process**:
- Test different edits
- See what works
- Learn from performance
- Iterate

**Result**: Better editing over time.
